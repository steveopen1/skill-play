# Skill Play - Codex Skills Collection

A curated collection of Codex CLI skills for enhanced AI agent capabilities.

## 📁 Skill Categories

### 🔒 Security Testing

**AI-Agent Security Architecture Test** (`skills/security-testing/ai-agent-security-architecture-test/`)

一个针对AI-Agent项目开展全流程对抗测试与优化的专业技能。

**核心能力：**
- 🎯 验证AI-Agent能力边界
- 🛡️ 提升安全对抗能力
- 🏢 企业级靶场构建
- 🔴 红队视角优化
- 📊 全流程结构化测试

**适用场景：**
- 渗透测试AI-Agent生产前安全评审
- XSS检测/SQL注入检测/漏洞扫描等功能迭代
- 代码审计Agent能力验证
- 权限控制检测Agent测试

**核心流程：**
1. Agent项目结构与能力分析
2. 目标能力预期清单生成
3. 企业级靶场测试文件构建
4. 测试结果评估（评审视角）
5. 红队视角优化方案制定
6. 多Agent并行任务分发
